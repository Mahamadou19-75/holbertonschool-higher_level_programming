Python More Classes
Description

Ce projet fait partie du cursus Holberton School – Higher Level Programming et a pour objectif de pratiquer la création de classes en Python.
On apprend à :

Définir des classes simples et avancées

Gérer les attributs privés et publics

Créer des getters et setters

Ajouter des méthodes pour manipuler et afficher les objets
